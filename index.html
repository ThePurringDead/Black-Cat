Welcome to Baruk On Air

This is a Baruk On Air app, created as a personal project for fun and testing.

Simple User Interface - Clean and intuitive UI for both mobile phones and Android TV.
M3U Playlist Support - Load channels from any M3U URL.
Channel List - Organized display of available channels with logos and groups.
Video Playback - HLS stream playback using ExoPlayer
